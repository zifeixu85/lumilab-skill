# 抖音 — 必做约束 + know-how

## 必做约束

```
1. 前 3 秒必有钩子（不达标直接跳过，完播率 0）
2. 口播脚本带停顿标记 [停顿] [快] [慢]
3. 竖屏 9:16（横屏视频会限流）
4. 标题 ≤ 14 字（左上角覆盖位置）
5. 话题标签 2-3 个（多于 5 个被识别为标签党）
6. 视频长度：1 分钟内最佳完播率，3 分钟内仍 OK，超 3 分钟流量梯度大
7. BGM：商业曲库选轻快版（避免触发版权）
8. 不要带"关注/点赞"等直白引导（限流）
```

## 前 3 秒钩子结构

5 种验证过的结构：

1. **数字冲击**："3 个月减 28 斤，我用的方法只有 1 个"
2. **反共识**："不要再 X 了，正确做法是 Y"
3. **悬念**："这个视频可能会让你删掉你的健身 App"
4. **痛点**："你每天熬夜，但你不知道这件事正在杀死你的免疫力"
5. **个人故事**："我做 X 行业 10 年，今天告诉你一个真相"

## 口播脚本格式

```
[钩子 0-3s]
"开场金句"
[停顿 0.5s]

[价值 3-15s]
"核心论点"，"详细展开"。
[快] 一个例子。

[结论 15-30s]
[慢] "所以，X 是 Y。"

[互动 30-45s]
"评论区告诉我..."
```

## 分镜（如有视频）

- 镜头不超过 5 个，每个 ≤ 10 秒
- 字幕全程在画面下方居中
- 字幕字号 = 视频宽度 / 12
- 字幕颜色：白色 + 黑色描边

## 标签策略

- 1 个大流量标签（如 #创业）
- 1-2 个精准标签（如 #独立开发者 #小红书运营）

## 2025–2026 规则更新

### 更新 1：完播率权重下调 + 「复看率 / 评论质量」上调（2025 Q3）

- **发布时间**：2025-08，巨量引擎《抖音流量分发逻辑白皮书 2025》
- **来源**：巨量学官方课程 + 抖音电商大会演讲摘要
- **变化**：
  - 完播率权重从历史的 ~40% 降到 ~20%，「复看率」「评论字数 / 评论互动率」「转发到群」三项合计权重升到 ~50%。
  - 5 秒钩子仍然重要，但 8–15 秒区段的「二次留人」成为新分水岭。
- **影响**：
  - 「短钩子 + 反转结尾」单一套路失效，必须设计中段悬念。
  - 评论区运营从加分项变成必做项。
- **应对建议**：
  - 视频中段（8–15 秒）插入「反问 + 二级承诺」让用户带着问题看到末尾。
  - 准备 5–10 条「钩子评论」开播 30 分钟内由小号顶起来，目标评论平均字数 ≥ 20。

### 更新 2：违禁词收紧 + 垂直度评分降级（2026 Q1）

- **发布时间**：2026-01，抖音安全中心 + 巨量引擎商家公告
- **来源**：抖音《社区自律公约 2026》+ 巨量引擎商业化政策更新
- **变化**：
  - 极限词清单扩容：「最」「第一」「唯一」之外，新增「秒杀」「亏本」「绝对」「永久」等约 50 个词；带这些词的视频会被限流不推送。
  - 垂直度评分（账号近 30 篇内容的标签集中度）权重上调，跨类目超过 2 个会触发「账号画像模糊」降级。
- **影响**：
  - 用 AI 批量生成的「话题热点跟拍」视频更容易触发降级。
  - 混合发美食 / 知识 / 带货的账号自然流被压制。
- **应对建议**：
  - 内容生成前过一遍违禁词词典（lumilab 可内置 词典）；用「同义稳妥词」替换：「最」→「靠前」/「亏本」→「让利」。
  - 一个账号锁定 1 个二级类目（如「家庭收纳」而非泛「生活」），跨类目内容开新号。
