# 微信公众号 — 必做约束 + know-how

## 必做约束

```
1. 标题 ≤ 22 字（钩子前置，超 22 字会被截断）
2. 外链转底部引用（不能内联，会被降权）
3. 封面 2.35:1（标准尺寸 900×383）
4. 段落 ≤ 80 字（移动端阅读节奏）
5. 配图 ≤ 8 张（多于 8 张影响打开率）
6. 「在看」位置 = 底部最后一段引导
7. 不能 24h 内删除重发（违规）
8. 不能发 ≥1 篇/天（订阅号限制）
```

## 标题写法

- **数字 + 时间**："3 个月 5 万粉的秘密"
- **反差**："我做了 10 年 X，最后发现 Y"
- **疑问**："为什么 X 在中国不流行？"
- **结论前置**："X 是 Y，不是 Z"

## 排版规则（baoyu-markdown-to-html 标准）

- 大标题 H1：18-22px，主题色
- 副标题 H2：16-18px，次级色
- 正文：14-16px，字间距 0-2px
- 段距：1.5-2em
- 引用块：左侧竖线 + 浅灰背景
- 代码块：等宽字体 + 浅色背景
- 图片：居中 + max-width 100%
- 行间距：1.7

## 外链处理（关键）

**禁用**：
- 内嵌 `<a>` 标签（会被替换为「点这里」+ 警告）
- 短链直接放正文

**推荐**：
- 文章最底部用「参考链接」段落，列出 URL
- 用「阅读原文」放唯一最重要链接
- 用「文末关键字回复」引导评论区导流

## 引导关注

- 文章开头**不**插入卡片（会降低正文打开率）
- 文章结尾插入"关注我，每周一篇"引导
- 文末配二维码（仅个人号）

## 2025–2026 规则更新

### 更新 1：订阅号信息流改版 + 视频号联动（2025 Q4）

- **发布时间**：2025-11，微信公众平台版本更新公告
- **来源**：微信公众平台后台「功能更新」+ 张小龙公开课摘要
- **变化**：
  - 订阅号入口从「时间序」彻底切到「兴趣推荐流」，关注但长期不读的账号会被折叠到二级入口。
  - 公众号文章可一键挂载视频号短视频，挂载后文章在「看一看」「视频号 feed」获得双倍曝光。
- **影响**：
  - 单纯「靠粉丝量自然打开」的逻辑失效，打开率下降到 1–3%，被推荐量反超关注量。
  - 没有视频号矩阵的账号会持续掉曝光。
- **应对建议**：
  - 每篇文章配 1 个 30–60 秒短视频挂载到视频号。
  - 标题前 14 字承担全部点击决策（信息流卡片只展示前 14 字），把数字 / 反共识 / 具体人名前置。

### 更新 2：付费阅读限制收紧 + 伪原创检测升级（2026 Q1）

- **发布时间**：2026-03，微信公众平台《创作者生态治理通告》
- **来源**：公众平台运营中心通告 + 「腾讯新闻」转载
- **变化**：
  - 付费阅读门槛上调：粉丝 ≥ 500 才可开通，单篇付费上限调到 208 元，且未开通原创的账号不可设付费。
  - 伪原创检测算法升级：跨账号搬运 / AI 洗稿 / 「同义词替换式改写」会被打上「低原创度」标签，影响推荐权重。
- **影响**：
  - 大量「付费拆解 / 付费报告」类账号需要重新攒粉丝才能恢复变现。
  - 直接用 AI 改写竞品文章的策略失效。
- **应对建议**：
  - 新账号前 30 篇专注原创积累，避开付费功能。
  - 用 AI 写「框架 + 草稿」，但事实 / 案例 / 数据必须人工补真实来源；引用他人内容时用官方「转载」机制而非洗稿。
