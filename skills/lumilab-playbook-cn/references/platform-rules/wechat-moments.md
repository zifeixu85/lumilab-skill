# 朋友圈 — 必做约束 + know-how

## 必做约束

```
1. ≤ 6 行（超过会被折叠"展开"，打开率掉 70%）
2. 首行钩子（折叠后只露 1-2 行）
3. 不含外链（朋友圈外链不能直接点击，导流用图片）
4. 配图 ≤ 9 张
5. 不能 24h 内发 ≥10 条（被识别为营销号）
6. 不能 ≥5 个 emoji 在一条朋友圈
```

## 6 行内容结构

```
Line 1: [钩子] 痛点 / 结论 / 情绪
Line 2: [展开 1] 一句话
Line 3: [展开 2] 一句话
Line 4: [展开 3] 一句话
Line 5: [转折] 一个反思 / 个人态度
Line 6: [CTA] "私聊聊" / "评论扣 1" / "想了解的 +V"
```

## 配图策略

- **1 张**：纯文字图（封面式，大字）
- **3 张**：1 主图 + 2 辅图（黄金比例）
- **6 张**：网格式，2 行 3 列
- **9 张**：完整 3×3 网格（适合分享流程/案例）

## 内容方向

| 类型 | 适合频率 |

## 2025–2026 规则更新

### 更新 1：企业微信 + 个人微信深度整合 + 24h 删除策略变化（2025 Q4）

- **发布时间**：2025-10，企业微信 5.0 版本更新
- **来源**：企业微信官方更新日志 + 腾讯智慧零售公开分享
- **变化**：
  - 企业微信朋友圈与个人微信朋友圈打通：企业微信发的内容可同步到客户的个人微信信息流，单客户每天最多看到 1 条企微内容。
  - 历史「3 天可见 / 半年可见」配置之外，新增「24 小时可见」选项，且系统会主动在 24h 后清理可见性（不再保留底片）。
- **影响**：
  - 私域运营彻底从「拉人头进群」转向「企微 + 朋友圈双触点」。
  - 用户对「过期消失」型内容的信任度提升，限时感真实化。
- **应对建议**：
  - 重要营销节点（新品 / 团购 / 直播预告）用企微号发，触达更稳定，避免被个人号风控。
  - 「限时 24h」类内容设置 24h 可见，文末写明「明天此时消失」，把平台特性变成内容钩子。

### 更新 2：社群导流降权 + 视频号 / 公众号互推白名单（2026 Q1）

- **发布时间**：2026-02，微信安全中心《社群治理通告》
- **来源**：微信团队公众号 + 企业微信运营白皮书
- **变化**：
  - 朋友圈出现「群二维码 / 入群链接 + 利益诱导文案」会被风控折叠（仅自己可见），重复触发 3 次封号 1–7 天。
  - 视频号 / 公众号互推不再受限：朋友圈分享视频号链接的曝光提升约 30%，分享公众号文章保持原权重。
- **影响**：
  - 直接发「扫码进群领资料」的旧玩法失效。
  - 微信内三端（朋友圈 / 视频号 / 公众号）联动权重明显高于纯朋友圈玩法。
- **应对建议**：
  - 入群引导改为「私聊 1V1 邀请」或「视频号留言区暗号」，不在朋友圈直发群码。
  - 朋友圈日更结构调整为：每天 1 条视频号引流 + 1 条公众号文章 + 1 条生活化内容，构建「三端共振」。
