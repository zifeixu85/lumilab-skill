Nº 01

市场快照

「想法验证」正在从「读方法论 + 手搭落地页」变成「AI 编排一条流水线」。独立开发者、副业人、OPC 的核心痛点不是缺工具，而是不知道在什么阶段调用什么、怎么串成一次完整验证。

规模信号

全球独立开发者与副业创业者数以千万计；国内「一人公司 / OPC」群体随 AI 工具普及快速扩大。idea-验证类工具搜索量稳定上升，但多为单通道、给评分、无下一步。

关键趋势
  • ◆ AI coding 工具（Lovable / v0 / Bolt）让「做产品」变快，反而放大了「该不该做」的验证缺口
  • ◆ 国内验证场景以内容平台为主（小红书 / 朋友圈 / 公众号），不是海外的纯网页 fake-door
  • ◆ 方法论（Mom Test / Lean Canvas / YC）供给充足，但散落在课程与书里，没有可执行的工具化封装
  • ◆ Skill / Agent 形态崛起，让「方法论 + 工程约束」可以固化成可调用的步骤
